## エピック1: 投票作成

### ストーリー1.1: 簡単な投票作成
**As a** チームリーダー  
**I want to** 簡単に投票を作成したい  
**So that** チームの意見を効率的に収集できる

**受け入れ条件**:
- [ ] タイトルと選択肢だけで投票を作成できる
- [ ] 3分以内に投票作成が完了できる
- [ ] 作成後すぐに投票URLが取得できる

### ストーリー1.2: 詳細設定付き投票作成
**As a** イベント主催者  
**I want to** 詳細な設定で投票を作成したい  
**So that** 参加者のニーズに合わせた投票ができる

**受け入れ条件**:
- [ ] 複数選択の有効/無効を設定できる
- [ ] 投票期限を設定できる
- [ ] カテゴリを設定して整理できる

## エピック2: 投票参加

### ストーリー2.1: 簡単な投票参加
**As a** 一般ユーザー  
**I want to** 簡単に投票に参加したい  
**So that** 手軽に意見を表明できる

**受け入れ条件**:
- [ ] ユーザー登録なしで投票できる
- [ ] ワンクリックで投票が完了する
- [ ] 投票後すぐに結果が確認できる

### ストーリー2.2: 投票結果の確認
**As a** 投票参加者  
**I want to** 投票結果をリアルタイムで確認したい  
**So that** 他の人の投票状況を把握できる

**受け入れ条件**:
- [ ] 投票後すぐに最新結果が表示される
- [ ] グラフで視覚的に結果が分かる
- [ ] 自分の投票内容が確認できる

## エピック3: 投票管理

### ストーリー3.1: 投票一覧の確認
**As a** 投票作成者  
**I want to** 作成した投票の一覧を確認したい  
**So that** 投票の状況を管理できる

**受け入れ条件**:
- [ ] 自分が作成した投票が一覧表示される
- [ ] 各投票の参加状況が分かる
- [ ] アクティブ/終了済みで分類される

### ストーリー3.2: 投票の共有
**As a** 投票作成者  
**I want to** 投票を簡単に共有したい  
**So that** より多くの人に参加してもらえる

**受け入れ条件**:
- [ ] URLを簡単にコピーできる
